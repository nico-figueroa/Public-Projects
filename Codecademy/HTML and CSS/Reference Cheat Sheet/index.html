CSS Font Properties

Reference Cheat Sheet

Font Property Property Description Coding Example
font-family Specifies the font family for an element font-family: Arial, sans-serif;
font-width Specifies the width of the font font-width: normal;
font-size Specifies the size of the font font-size: 16px;
font-size-adjust Specifies the aspect value for an element font-size-adjust: 0.5;
font-language-override Specifies the language override for an element font-language-override: "en-US";
font-weight Specifies the weight (boldness) of the font font-weight: bold;
font-style Specifies the style of the font (normal, italic, oblique) font-style: italic;
font-feature-settings Specifies the font feature settings for an element font-feature-settings: "kern" 1;
font-kerning Specifies the kerning behavior for an element font-kerning: auto;
font-palette Specifies the font palette for an element font-palette: "kern" 1;
font-smooth Specifies the font smoothing behavior for an element font-smooth: auto;
font-stretch Specifies the stretch behavior for an element font-stretch: normal;
font-optical-sizing Specifies the optical sizing behavior for an element font-optical-sizing: auto;
font-synthesis Specifies the font synthesis behavior for an element font-synthesis: auto;
font-synthesis-position Specifies the position of the synthesized font font-synthesis-position: auto;
font-synthesis-small-caps Specifies whether small-caps synthesis is allowed font-synthesis-small-caps: auto;
font-synthesis-style Specifies whether italic or oblique synthesis is allowed font-synthesis-style: auto;
font-synthesis-weight Specifies whether bold synthesis is allowed font-synthesis-weight: auto;

font-variant-numeric Specifies the numeric glyphs for an element font-variant-numeric: normal;
