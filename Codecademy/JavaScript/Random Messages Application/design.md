# Stellar travel generator

## Codecademy Input
---------------------------------

**Requirements:**

1. Build a message generator program. 
2. Every time a user runs a program, they should get a new, randomized output. 
3. You’re welcome to take the project in a couple of different forms, like an astrology generator, inspirational message, or nonsensical jokes. 
4. To make your program truly random, the message that it outputs should be made up of at least three different pieces of data. 
5. Take what you know of JavaScript syntax so far to build the program and customize it to your liking.

**Project Objectives:**
* Build a message generator program using JavaScript
* Use Git version control
* Use command line
* Develop locally on your computer

**Prerequisites:**
* JavaScript
* Git and GitHub
* Command line

## Design
---------------------------------

**Concept:**

Generate a message that shares the spaceship to use, the distance to travel and the destination to arrive to.

**High Level Code Architecture:**

* 1 Javascript file with a single word name for easy use
    * engage.js
* 2 arrays for the message components: 
    1. Spaceships
    2. Destinations
* 3 functions to create and display the message:
    * Component index selector
    * Message builder
    * Message presentation

**Development process**

1. Generate a new git branch for my project (engage)
2. Describe the design in a design.md file
3. Code the application in engage.js within VS Code and commit after completion of every project file
4. Test the application
5. Use the VS Code Git Hub terminal to push code to the remote repository
6. Generate a pull request in the Git Hub repository and approve it
7. Merge the changes to main
